# Master SQL — Learn SQL by Doing

Hands-on SQL on the real **Northwind** dataset, taught on **PostgreSQL** with portable
ANSI SQL (vendor differences flagged).

**The promise:** you *write* the queries yourself, against real business data — guess, run,
understand. Not slides you watch.

## What's covered

Three series, a natural progression. This is early — lessons are still being built out, so
the outline below is the direction, not a fixed contract.

**Series 1 — Fundamentals.** Query one table, then glue two: SELECT, WHERE, ORDER BY,
DISTINCT & aliases, the aggregates, GROUP BY, JOINs, NULLs — ending in your first real report.

**Series 2 — Intermediate.** Answer real business questions: multi-table joins, HAVING,
self-joins, subqueries, CASE, dates, text & regex search, set operations, and views.

**Series 3 — Advanced.** Production-grade and cross-vendor: window functions, CTEs &
recursive CTEs, pivots, cohorts, reading query plans, indexing, and portable SQL that runs
the same on every engine.
